# Garage-Heater-Thermostat-Mod
Files and information pertaining to the video "Wi-Fi Enabled Thermostat On An Electric Garage Heater - DIY" - https://youtu.be/Dj_aC5-Oeg0

Two files are included:
1.  Garage Heater Thermostat Cable Strain Relief - Plug.stl
2.  Garage Heater Thermostat Cable Strain Relief- Cap.stl

These are used as strain-relief cable protectors for the thermostat cable entering the heater unit.
